# md5-crypt
Node.js md5 console cryptor &amp; decryptor

## Install libraries
Run `npm i` command

## How to Use
1. Run application: `node crypt.js` and follow instructions
2. Run application with keys and data: `node crypt.js --enc data` or `node crypt.js --dec data`

## Dependencies
- *"md5"*: `2.2.1`
- *"reverse-md5"*: `0.0.4`
